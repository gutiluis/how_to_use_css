International Web Accessibility Standards

Inernational Web Content Accessibility Guidelines (Latest Version 2.2)

Web Content Accessibility Guidelines (WCAG):

  • Risk prevention and exposure to lawsuits.
  • Americans with Disabilities Act.
  • Disability rights are civil rights.
  • Make content accessible to users with dasbilities.
  • Accessibility laws.

Principles of Accessibility:

  • Perceivable
  • Operable
  • Understandable
  • Robust

Accessibility Guidelines:

Perceivable

Time-Based Media

Alternatives for time-based media.

  • Captions (Live) Level AA
  • Audio Description (Prerecorded) Level AA
  • Sign Language (Prerecorded) Level AAA
  • Extended Audio Description (Prerecorded) Level AAA
  • Media Alternative (Prerecorded) Level AAA
  • Audio-only (Live) Level AAA
Adaptable

Contente presentable in different ways. A simpler layout without losing information or structure.

  • Orientation Level AA
  • Identify Input Purpose Level AA
  • Identify Purprose Level AAA
Distinguishable

Easier for users to see and hear content including separating foreground from background.

  • Contrast (Minimum) Level AA
  • Resize Text Level AA
  • Images of Text Level AA
  • Contrast (Enhanced) Level AAA
  • Low or No Background Audio Level AAA
  • Visual Presentation Level AAA
  • Images of Text (No Exception) Level AAA
  • Reflow Level AA
  • Non-text Contrast Level AA
  • Text Spacing Level AA
  • Content on Hover or Focus Level AA

Operable

Keyboard Accessible

Make all functionality available from a keyboard.

  • Keyboard (No Exception) Level AAA
Enough Time

Provide users enough time to read and use content.

  • No Timing Level AAA
  • Interruptions Level AAA
  • Re-authenticating Level AAA
  • Timeouts Level AAA
Seizures and Physical Reactions

Do not design content in a way that is known to cause seizures or physical reactions.

  • Three Flashes Level AAA
  • Animation from Interactions Level AAA
Navigable

Ways to help users navigate, find content, and determine where they are.

  • Multiple Ways Level AA
  • Headings and Labels Level AA
  • Focus Visible Level AA
  • Location Level AAA
  • Link Purpose (Link Only) Level AAA
  • Section Headings Level AAA
Input Modalities

Make it easier for users to operate functionality through various inputs beyond keyboard.

  • Target Size (Enhanced) Level AAA
  • Concurrent Input Mechanisms Level AAA
  • Dragging Movements Level AA
  • Target Size (Minimum) Level AA

Understandable

Readable

Make text content readable and understandable.

  • Language of Parts Level AA
  • Unusual Words Level AAA
  • Abbreviations Level AAA
  • Reading Level Level AAA
  • Pronunciation Level AAA
Predictable

Make web pages appear and operate in predictable ways.

  • Consistent Navigation Level AA
  • Consistent Identification Level AA
  • Change on Request Level AAA
Input Assistance

Help users avoid and correct mistakes.

  • Error Suggestion Level AA
  • Error Prevention (Legal, Financial, Data) Level AA
  • Help Level AAA
  • Error Prevention Level AAA

Robust

Compatible

Maximize compatibility with current and future user agents, including assistive technologies.

  • Status Messages Level AA
